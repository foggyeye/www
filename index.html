Case Studies

Continuous Delivery pipeline

Technologies Jenkins, Groovy, Git, Github, Ruby

As part of the team for one of our clients, we were tasked with setting up a continuous delivery pipeline, which enabled the client to rapidly deploy new versions of their product to different environments, accessed by different members of the team at varying intervals. The pipeline was setup to ensure that each commit to the source code repository triggered the pipeline’s workflow, which would create a packaged, tested and production deployable artefact. This immutable deployable artefact could then be promoted to different environments at the click of a button.

Data workflow engine

Technologies Hadoop, Ruby

One of our clients required a workflow engine that could be used to manage a Hadoop workflow. Non of the existing tools / frameworks available at the time of building the workflow engine met the client's requirements so we embarked on building a testable and production ready workflow application. The use of the application meant that the team had a consistent way of running the workflow engine, which was run multiple times daily in the production and non-production environments. The application was an integral part of an overall system responsible for processing over six million data points a day.
